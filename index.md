# Building an AI Coding Agent

**A 4-hour hands-on workshop where participants build a working AI coding agent in Python.**

---

## Schedule

| Time | Session | Format |
|------|---------|--------|
| 0:00 – 0:20 | What is an AI Agent? + Live Demo | Talk |
| 0:20 – 0:50 | Architecture Deep Dive | Talk |
| 0:50 – 1:10 | Context Window & Model Selection | Talk |
| 1:10 – 1:30 | System Prompts & Safety | Talk |
| 1:30 – 1:40 | **Break** | ☕ |
| 1:40 – 2:00 | Starter Code Walkthrough + Q&A | Talk |
| 2:00 – 4:00 | Hands-On Coding (4 milestones) | Workshop |

---

## What Participants Need

- A **laptop** with a code editor installed
- **Python 3.10+** installed
- An **Anthropic API key** — sign up free at [console.anthropic.com](https://console.anthropic.com) (the free tier is sufficient, no payment needed)
- **Internet access** at the venue

---

## What the Venue / Organiser Should Arrange

- **Reliable Wi-Fi** for all attendees (everyone will be making API calls)
- **Power strips / extension cords** — participants need to charge laptops
- **Projector and screen** for the speaker
- **Wi-Fi credentials** printed or displayed prominently
- **Printed handouts** (2-3 pages) — provided in the [workshop repo](https://github.com/chandrameenamohan/iit-acb-talk)
